# 🎬 Video Downloader

Um aplicativo simples e eficiente para download de vídeos diretamente da internet.

## 🚀 Sobre o projeto

O **Video Downloader** é uma aplicação desenvolvida com foco em praticidade, permitindo que usuários façam download de vídeos a partir de links de forma rápida e intuitiva.

O projeto utiliza a biblioteca **pytubefix**, uma alternativa atualizada para download de vídeos, garantindo maior estabilidade.

---

## 🛠️ Tecnologias utilizadas

- Python 🐍
- pytubefix
- FFmpeg (opcional, para conversão de mídia)

---

## ⚙️ Funcionalidades

- 📥 Download de vídeos via URL
- 🎧 Download de áudio
- 🎞️ Escolha de qualidade
- 📂 Salvamento automático local
- ⚡ Execução simples via terminal

---

## 📦 Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/BrunoSouzz/videodownloader.git
```
